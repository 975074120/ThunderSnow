      启动绑定模块失败

      sudo mkdir /mnt/自定义名称

      sudo mount --bind 绑定地址 /mnt/自定义名称

      比如:（把你个人目录下的Download文件夹绑定到迅雷下载地址）

      sudo mkdir /mnt/thunder

      sudo mount --bind /home/$USER/Download/ /mnt/thunder/

      PS:迅雷不会认识thunder这个名字，只能按顺序识别成C，D，E盘
